## Getting started

Requirements:
 - Dart SDK >=2.18.2
 - Flutter SDK >= 3.3.0

Add ```barberia_ui``` as a dependency in your pubspec.yaml file.
```yaml
dependencies:
  barberia_ui:
    git: https://github.com/kevinkwee/barberia_ui.git
    ref: main
```
